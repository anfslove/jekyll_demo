配置仿真的虚拟主机开发环境

  • 打开hosts文件(c:/windows/system32/drivers/etc/hosts)，加入如下代码
    127.0.0.1 www.abc.cn
    192.168.101.253 www.abc.cn
    127.0.0.1 localhost
    127.0.0.1 www.abc.cn
    127.0.0.1 www.tdeast.cn
  • 打开apache的httpd.conf文件,找到
    #Virtual hosts
    #Include conf/extra/httpd-vhosts.conf

    修改为

    #Virtual hosts
    Include conf/extra/httpd-vhosts.conf
  • 打开httpd-vhosts.conf文件 {% highlight xml %} DocumentRoot "D:/apache/htdocs" ServerName localhost DirectoryIndex index.html index.php AllowOverride None Options None Order allow,deny Allow from all DocumentRoot "D:/apache/htdocs/abc" ServerName www.abc.cn DirectoryIndex index.html index.php AllowOverride None Options None Order allow,deny Allow from all DocumentRoot "F:/work/tdeast" ServerName www.tdeast.cn DirectoryIndex index.html index.php AllowOverride None Options None Order allow,deny Allow from all {% endhighlight %}
