配置仿真的虚拟主机开发环境

  • 打开hosts文件(c:/windows/system32/drivers/etc/hosts)，加入如下代码
    127.0.0.1 www.abc.cn
    192.168.101.253 www.abc.cn
    127.0.0.1 localhost
    127.0.0.1 www.abc.cn
    127.0.0.1 www.tdeast.cn
  • 启用Apache的虚拟主机功能
    #LoadModule vhost_alias_module modules/mod_vhost_alias.so
    打开apache的httpd.conf文件,找到
    #Virtual hosts
    #Include conf/extra/httpd-vhosts.conf

    修改为

    #Virtual hosts
    Include conf/extra/httpd-vhosts.conf
  • 打开httpd-vhosts.conf文件 {% highlight xml %} NameVirtualHost 115.28.183.202:80 DocumentRoot C:/Projects/dcxc ServerName 115.28.183.202 ServerAlias 115.28.183.202 {% endhighlight %}
