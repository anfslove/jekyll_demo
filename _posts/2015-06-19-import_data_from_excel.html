DISABLE TRIGGER ALL' EXEC sp_MSForEachTable 'DELETE FROM ?' after import data EXEC sp_MSForEachTable 'ALTER TABLE ? CHECK CONSTRAINT ALL' EXEC sp_MSForEachTable 'ALTER TABLE ? ENABLE TRIGGER ALL' EXEC sp_MSFOREACHTABLE 'SELECT * FROM ?' 4.Import data from the excel
